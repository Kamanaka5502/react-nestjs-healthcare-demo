Full-Stack Demo — React + NestJS + TypeScript + SQL + GraphQL

This project demonstrates a modern full-stack application built using React, NestJS, TypeScript, and a SQL database. It includes REST and GraphQL endpoints, responsive UI components, and clean backend architecture.

Tech Stack

Frontend: React, TypeScript, Axios, React Router

Backend: NestJS, TypeScript, REST, GraphQL (code-first)

Database: PostgreSQL (SQL schema + seed data)

Tooling: Docker Compose, Git, npm/yarn



---

Features

Responsive React UI

SQL-backed NestJS API with proper modules, services, controllers

GraphQL endpoint (/graphql) with simple user schema

REST endpoint (/api/users) for comparison

Clean architecture with DTOs, dependency injection, and validation

Fully typed TypeScript across frontend and backend



---

Setup

1. Clone repo

git clone https://github.com/YOUR_USERNAME/react-nestjs-healthcare-demo
cd react-nestjs-healthcare-demo

2. Start backend + database

cd server
npm install
npm run start:dev

3. Start frontend

cd client
npm install
npm start

App runs at:
Frontend → http://localhost:3000
Backend → http://localhost:4000
GraphQL → http://localhost:4000/graphql
